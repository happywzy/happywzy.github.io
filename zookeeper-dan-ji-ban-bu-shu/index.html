zookeeper单机版部署

发表于 2020-03-03 更新于 2020-04-16 分类于 tools

下载压缩包

wget http://mirror.bit.edu.cn/apache/zookeeper/zookeeper-3.5.7/apache-zookeeper-3.5.7-bin.tar.gz

tar -zxvf apache-zookeeper-3.5.7-bin.tar.gz 

cd apache-zookeeper-3.5.7-bin/

注意：zookeeper从3.5.5版本开始，需要下载*-bin.tar.gz的包才能运行，否则启动会报错，缺少类。

拷贝zoo.cfg

用默认配置即可

cp conf/zoo_sample.cfg conf/zoo.cfg

启动&检查

启动

./bin/zkServer.sh start

检查

./bin/zkServer.sh status

停止

./bin/zkServer.sh stop

重启

./bin/zkServer.sh restart
